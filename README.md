# Erin Countdown

A simple shared countdown app for Erin and Evan, counting down to **March 22, 2026**.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build check

```bash
npm run lint
npm run build
```

## Update the reunion date

Edit `src/app/page.tsx` and change the `TARGET_DATE` constant.
